### 泡泡 v4 — 2 秒空闲冻结 + 尺寸增大
- `BASE_R` 从 85 → 100
- 新增 `lastInteractRef` 和 `frozenRef`：任意鼠标事件（mousedown/mousemove/mouseup）通过 `markInteract()` 重置时间戳并解冻

- 初始化时 `lastInteractRef = performance.now()` 防止挂载瞬间误冻结

### 泡泡 v5 — 精简 + Rolling 重构
- 删除自驱动力（wobble）和 2 秒空闲冻结系统
- **Rolling 按钮位置**：从头部的标题下方移至 `.bubbles__canvas-wrap` 右下角（`position: absolute; bottom: 16px; right: 20px`），带半透明暗底 + `backdrop-filter: blur` 毛玻璃效果
- **Rolling 功能重构**：不再重新生成泡泡位置，改为给每个泡泡在当前位置赋予随机方向的初速度（`Math.random() * 2π` 方向 + `MAX_SPEED * 0.6~1.8` 速度量级 + 轻微向上偏置），附带缩放和碰撞闪光

### 泡泡 v6 — 翻滚（Tumbling）效果
- 每个泡泡新增 `rotation`（角度）和 `rotV`（角速度）字段，在 Canvas 中 `ctx.translate + ctx.rotate` 绘制
- 视觉旋转指示：文字随泡泡旋转 + 泡泡顶部边缘小白点（rotation indicator dot）让旋转清晰可见
- **角动量物理**：
  - 碰撞时传递切向速度 → 角速度（`tangentSpeed * 0.15`）
  - 墙壁/地板反弹时根据碰壁速度触发旋转（`rotV += vy * 0.002` 等）
  - 拖拽时水平移动自动产生旋转（`rotV += vx * 0.003`）
  - 独立角速度阻尼 `ROT_DAMPING = 0.993`，比线速度衰减略快
  - 角速度上限 `MAX_ROT = 0.04 rad/frame`
- **Rolling 按钮**：给每个泡泡强角速度（`±MAX_ROT * 2.2`）+ 随机方向初速度（`MAX_SPEED * 1.0~2.6`）+ 强力向上弹射（vy - 1.2）
